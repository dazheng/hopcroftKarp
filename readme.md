# Hopcroft-Karp
二分图最大匹配Hopcroft-Karp算法go语言实现。是将 https://github.com/sofiatolaosebikan/hopcroftkarp 中的python实现改为go实现。

## Installation

```
go get -u github.com/dazheng/hopcroftkarp
```
